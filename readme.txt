Simple bookmarket to open the wp-admin of the current site in a new tab. Never type wp-admin ever again.
If it is not a Wordpress site, you just get a 404.
Works on localhost Wordpress sites.

Add it to your browser:
1. In wp-admin-bookmarklet.js, copy the single-line javascript code at the top of the file.
2. Create a new bookmark in your browser.
3. Paste code in the URL/location field.
4. Whatever name you like in the name field, e.g. "wp-admin".
5. Click Add/Save/OK/whatever your browser says in the affirmative.
